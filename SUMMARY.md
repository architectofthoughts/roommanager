# 방 매니저 — 빠른 참조 가이드

---

## 이 앱은 무엇인가요?

- 브라우저에서 동작하는 **방/공간 관리 웹앱**입니다.
- 가구를 캔버스에 배치하고, 각 가구에 물품을 등록하여 관리합니다.
- 모든 데이터는 브라우저에 자동 저장됩니다.

---

## 설치 방법

### 필요한 것

- Node.js 20 이상
- npm

### 설치 순서

```bash
git clone https://github.com/architectofthoughts/roommanager.git
cd roommanager
npm install
cp .env.example .env    # Gemini API 키 설정 (선택)
npm run dev             # http://localhost:5173 에서 실행
```

### Windows 사용자

- `run.bat` 더블클릭 — 설치부터 실행까지 자동

---

## 핵심 기능 요약

### 가구 관리

- 좌측 사이드바에서 모양/카테고리/이름 지정 후 추가
- 캔버스에서 드래그로 이동 (0.5셀 단위 스냅)
- 핸들로 크기 조절, 회전 (45도 스냅)
- 우측 사이드바에서 색상, 외곽선(실선/점선/없음, 두께, 색상), 메모 편집

### 물품 관리

- 가구 선택 후 우측 사이드바에서 물품 추가
- 이름, 수량, 카테고리, 층(1~10), 메모 입력
- 물품별 수정/삭제 가능

### 방 크기 조절

- 상단 바에서 가로/세로 그리드 수 직접 입력 (4~50)

### AI 사진 분석

- 상단의 "사진 분석" 버튼 클릭
- 이미지 업로드 후 "분석하기" 클릭
- AI가 물품을 인식하여 목록 생성
- 원하는 항목을 선택하여 가구에 일괄 등록
- API 키 없이도 데모 모드로 체험 가능

### 통계

- 상단의 "통계" 버튼 클릭
- 가구 수, 물품 종류, 총 수량 요약
- 카테고리별 분포 막대 그래프
- 가구별 보관 현황

### 검색

- 상단 검색창에서 물품명/카테고리/메모 통합 검색

## 꼭 알아야 할 것

- **저장**: 자동 저장 (localStorage). 별도 저장 버튼 없음.
- **데이터 주의**: 브라우저 데이터 삭제 시 함께 삭제됨.
- **마이그레이션**: 이전 버전 데이터는 자동으로 변환됨.
- **Gemini API**: `.env`에 키 설정 필요. 없으면 데모 모드.
- **화면**: 최소 1024px 폭. 데스크톱 최적화.

## 문제 해결

| 증상 | 해결 |
|------|------|
| 앱이 안 열림 | Node.js 20+ 설치 확인, `npm install` 실행 |
| run.bat 오류 | Node.js가 PATH에 있는지 확인 |
| 데이터 사라짐 | 브라우저 데이터 삭제 여부 확인 |
| AI 분석 안 됨 | `.env` 파일의 API 키 확인 후 서버 재시작 |
