# 방 매니저 사용 매뉴얼

이 문서는 방 매니저(RoomManager) 애플리케이션의 사용 방법을 단계별로 안내합니다.

---

## 1. 시작하기

### 처음 설치 (WSL/Mac/Linux)

```bash
# 1. 저장소 클론
git clone https://github.com/architectofthoughts/roommanager.git
cd roommanager

# 2. 의존성 설치
npm install

# 3. 환경 변수 설정 (선택사항 — AI 분석 기능을 사용하려면 필요)
cp .env.example .env
# .env 파일을 열어 VITE_GEMINI_API_KEY 값을 입력

# 4. 개발 서버 실행
npm run dev
```

브라우저에서 `http://localhost:5173`을 열면 앱을 사용할 수 있습니다.

### 처음 설치 (Windows)

1. 프로젝트 폴더에서 `run.bat` 파일을 더블클릭합니다.
2. Node.js가 설치되어 있지 않다면 [Node.js 공식 사이트](https://nodejs.org/)에서 먼저 설치합니다.
3. `run.bat`이 자동으로 의존성 설치와 개발 서버 실행을 처리합니다.
4. 터미널에 표시되는 URL(`http://localhost:5173`)을 브라우저에서 엽니다.

### Gemini API 키 설정 (선택)

AI 사진 분석 기능을 사용하려면 Google Gemini API 키가 필요합니다.

1. [Google AI Studio](https://aistudio.google.com/apikey)에 접속합니다.
2. API 키를 생성합니다 (무료).
3. 프로젝트 루트의 `.env` 파일을 열고 다음과 같이 입력합니다:
   ```
   VITE_GEMINI_API_KEY=여기에_API_키_입력
   ```
4. 개발 서버를 재시작합니다.

> API 키가 없어도 앱은 정상 동작합니다. 사진 분석 시 데모 데이터가 표시됩니다.

## 2. 화면 구성

앱은 크게 네 영역으로 구성됩니다:

```
┌────────────────────────────────────────────────────┐
│                   상단 바 (TopBar)                    │
├──────────┬─────────────────────────┬───────────────┤
│          │                         │               │
│  좌측     │     캔버스 (RoomCanvas)   │    우측       │
│ 사이드바   │                         │   사이드바     │
│          │                         │               │
├──────────┴─────────────────────────┴───────────────┤
```

### 상단 바

- **방 이름**: 클릭하여 이름 변경 가능 (Enter로 확정, Esc로 취소)
- **방 크기**: 가로/세로 그리드 수를 직접 입력하여 조절 (4~50 범위)
- **검색**: 물품명, 카테고리, 메모로 통합 검색
- **통계 버튼**: 물품 통계 대시보드 열기
- **사진 분석 버튼**: AI 사진 분석 모달 열기

### 좌측 사이드바

- 가구 추가 폼 (모양, 카테고리, 이름)
- 전체 가구 목록

### 캔버스

- 가구를 시각적으로 배치하는 인터랙티브 영역
- 드래그로 이동, 핸들로 크기 조절, 회전 가능

### 우측 사이드바

- 선택한 가구의 상세 정보 편집
- 해당 가구에 수납된 물품 목록 및 관리

---

## 3. 가구 관리

### 가구 추가하기

1. 좌측 사이드바에서 **모양 선택** (사각형 / 원형)
2. **카테고리 선택** (수납장, 침대, 테이블, 의자, 가전, 기타)
3. **이름 입력** 후 "추가" 버튼 클릭 또는 Enter
4. 캔버스에 가구가 나타나고 자동으로 선택됩니다.

### 가구 이동하기

- 캔버스에서 가구를 **드래그**하여 이동합니다.
- 이동 시 **0.5셀 단위**로 스냅되어 정밀한 배치가 가능합니다.

### 가구 크기 조절하기

- 가구를 선택하면 나타나는 **변환 핸들**(모서리/변의 점)을 드래그하여 크기를 조절합니다.
- 최소 1셀 크기까지 줄일 수 있습니다.

### 가구 회전하기

- 캔버스에서 가구를 선택한 후 **회전 핸들**(상단의 원형 점)을 드래그합니다.
- 45도 단위로 스냅됩니다 (0, 45, 90, 135, 180, 225, 270, 315).
- 또는 우측 사이드바의 **회전 슬라이더**로 15도 단위로 조절 가능합니다.

### 가구 속성 편집하기

가구를 선택하면 우측 사이드바에서 다음을 편집할 수 있습니다:

- **이름**: 텍스트 입력
- **카테고리**: 드롭다운 선택
- **색상**: 10가지 프리셋 색상 중 선택
- **외곽선**: 스타일(실선/점선/없음), 두께(0.5~5), 색상
- **회전**: 슬라이더로 각도 조절
- **메모**: 자유 텍스트 입력

### 가구 외곽선 편집하기

1. 가구를 선택한 뒤 우측 사이드바의 **"외곽선"** 섹션을 확인합니다.
2. **스타일 선택**: 실선(solid) / 점선(dashed) / 없음(none)
3. 실선 또는 점선을 선택한 경우:
   - **두께 슬라이더**: 0.5~5 범위에서 0.5 단위로 조절
   - **색상 피커**: 외곽선 색상을 자유롭게 지정
4. 캔버스에 실시간으로 반영됩니다.

### 가구 삭제하기

1. 가구를 선택합니다.
2. 우측 사이드바 하단의 **"가구 삭제"** 버튼을 클릭합니다.
3. 해당 가구에 수납된 물품이 있다면 확인 대화상자가 나타납니다.

## 4. 물품 관리

### 물품 추가하기

1. 물품을 추가할 **가구를 선택**합니다.
2. 우측 사이드바 하단의 **"물품 추가"** 영역에서:
   - **물품 이름** 입력
   - **수량** 입력
   - **카테고리** 선택 (의류, 책, 전자기기, 식품, 생활용품, 문구, 주방용품, 기타)
   - **층** 선택 (1층~10층) — 가구 내 수납 위치를 층으로 구분
   - **메모** 입력 (선택)
3. "추가" 버튼 클릭 또는 이름 입력 칸에서 Enter

### 물품 수정하기

1. 물품 목록에서 해당 물품 우측의 **연필 아이콘**을 클릭합니다.
2. 이름, 수량, 카테고리, 층, 메모를 수정합니다.
3. "완료" 버튼을 클릭합니다.

### 물품 삭제하기

- 물품 목록에서 해당 물품 우측의 **휴지통 아이콘**을 클릭합니다.

### 물품의 층(Floor) 정보

- 각 물품에 1~10층까지의 **층 정보**를 설정할 수 있습니다.
- 물품 목록에서 층 번호가 배지 형태로 표시됩니다 (예: "3층").
- 수납가구의 선반이나 서랍의 위치를 직관적으로 구분할 때 유용합니다.

---

## 5. 방 크기 조절

1. 상단 바에서 그리드 아이콘 옆의 **숫자 입력 필드**를 찾습니다.
2. **가로** 그리드 수와 **세로** 그리드 수를 직접 입력합니다.
3. 유효 범위: 4 ~ 50 (그리드 단위)
4. 변경 즉시 캔버스에 반영되며, 자동 저장됩니다.

> 방 크기를 줄이면 기존 가구가 캔버스 밖으로 나갈 수 있습니다. 필요시 가구를 먼저 이동해주세요.

---

## 6. AI 사진 분석

### 사진 분석 실행하기

1. 상단 바의 **"사진 분석"** 버튼을 클릭합니다.
2. 이미지를 **드래그 & 드롭**하거나 **클릭하여 업로드**합니다. (JPG, PNG, WEBP 지원)
3. 업로드 후 **"분석하기"** 버튼을 클릭합니다.
4. AI가 사진 속 물품을 인식하여 목록을 생성합니다.

### 분석 결과 활용하기

1. 분석 결과에서 각 물품의 **이름, 수량, 카테고리**를 확인하고 필요시 수정합니다.
2. 체크박스로 추가할 물품을 선택/해제합니다.
3. 하단의 **"대상 가구"** 드롭다운에서 물품을 넣을 가구를 선택합니다.
4. **"선택 항목 추가"** 버튼을 클릭하면 선택된 물품이 해당 가구에 일괄 등록됩니다.

> API 키 미설정 시 "데모 모드"로 동작하며, 샘플 물품(노트북, 충전기, 책 등)이 표시됩니다.

---

## 7. 물품 통계

1. 상단 바의 **"통계"** 버튼을 클릭합니다.
2. 대시보드에서 확인할 수 있는 정보:
   - **요약 카드**: 가구 수, 물품 종류 수, 총 수량
   - **카테고리별 분포**: 물품 카테고리별 수량 막대 그래프
   - **가구별 보관 현황**: 가구별 물품 수량, 종류, 포함 카테고리

---

## 8. 검색

1. 상단 바의 **검색창**에 검색어를 입력합니다.
2. 물품명, 카테고리, 메모에서 해당 문자열이 포함된 물품이 필터링됩니다.
3. X 버튼으로 검색어를 초기화할 수 있습니다.

---

## 9. 데이터 저장 및 마이그레이션

- 모든 데이터는 **브라우저 localStorage**에 자동 저장됩니다. (키: `roommanager-data`)
- 별도의 저장 버튼이 없으며, 모든 변경사항이 **즉시** 저장됩니다.
- 이전 버전에서 저장된 데이터는 앱 로드 시 **자동 마이그레이션**됩니다:
  - 가구의 `borderStyle`, `borderWidth`, `borderColor` 필드가 기본값으로 추가됨
  - 물품의 `floor` 필드가 기본값 1로 추가됨

## 10. 트러블슈팅

### 앱이 열리지 않아요
- Node.js 20 이상이 설치되어 있는지 확인하세요: `node --version`
- 의존성이 설치되어 있는지 확인하세요: `npm install`
- 포트 5173이 다른 프로세스에 의해 사용 중이 아닌지 확인하세요.

### Windows에서 run.bat 실행 시 오류가 발생해요
- Node.js가 시스템 PATH에 등록되어 있는지 확인하세요.
- 명령 프롬프트에서 `node --version`이 출력되는지 확인하세요.
- 안 되면 Node.js를 재설치하고 "Add to PATH" 옵션을 선택하세요.

### 데이터가 사라졌어요
- 브라우저의 "사이트 데이터 삭제" 또는 "캐시 삭제"를 수행하면 localStorage 데이터가 삭제됩니다.
- 시크릿(비공개) 모드에서는 브라우저를 닫으면 데이터가 사라집니다.

### AI 분석이 작동하지 않아요
- `.env` 파일에 `VITE_GEMINI_API_KEY` 값이 올바르게 설정되어 있는지 확인하세요.
- 개발 서버를 재시작하세요 (환경 변수 변경 후 재시작 필요).
- API 키가 유효한지 [Google AI Studio](https://aistudio.google.com/apikey)에서 확인하세요.

### 이전 데이터가 깨져 보여요
- v2에서 추가된 필드(외곽선, 층 정보)는 자동 마이그레이션됩니다.
- 그래도 문제가 있다면 브라우저 개발자 도구 > Application > Local Storage에서 `roommanager-data` 키의 데이터를 확인하세요.

## FAQ

**Q: 데이터를 다른 브라우저로 옮길 수 있나요?**
A: 현재 내보내기/가져오기 기능은 없습니다. 브라우저 개발자 도구에서 localStorage의 `roommanager-data` 값을 수동으로 복사할 수 있습니다.

**Q: 여러 방을 관리할 수 있나요?**
A: 현재는 하나의 방만 관리할 수 있습니다. 방 이름은 변경 가능합니다.

**Q: 가구를 캔버스 밖으로 드래그하면 어떻게 되나요?**
A: 가구는 캔버스 밖으로도 이동 가능합니다. 방 크기를 조절하거나 가구를 다시 안으로 드래그하세요.

**Q: 모바일에서 사용 가능한가요?**
A: 최소 화면 폭이 1024px로 설정되어 있어 데스크톱 환경에 최적화되어 있습니다. 태블릿 가로 모드에서는 사용 가능할 수 있습니다.
